# Shareboard

## Intro

OOP PHP learning project using a mini framework and MVC.

## Requirements

- PHP7
- PHP-PDO
- PDO-mysql
- MySQL or MariaDB server
- Apache configured with homedir and rewrite module
- Bootstrap4 (at least the CSS file)
